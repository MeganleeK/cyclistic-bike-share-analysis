# Cyclistic Bike Share Analysis

## Overview
This project analyzes Cyclistic bike-share data to identify behavioral differences between annual members and casual riders and provide recommendations to increase membership conversions.

## Tools Used
- SQL Server
- Power BI

## Business Question
How can Cyclistic convert casual riders into annual members?

## Key Findings
- Casual riders take longer rides than members.
- Casual riders frequently use electric bikes.
- Casual riders generated slightly more rides during June 2021.

## Recommendations
- Develop recreational membership packages.
- Offer electric-bike incentives.
- Target active casual riders with membership campaigns.
